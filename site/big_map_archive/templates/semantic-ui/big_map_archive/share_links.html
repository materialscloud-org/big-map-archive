Share links

Create links to give access to view a record, view a draft (preview) or edit a draft.
Note that a link gives access with the selected permission to any version of a record.
Only the owner of the record can share the record with a community and can create or delete the record access links.

Can view - access to view a record

To create a link with permission to view a record, the record should have been already shared with a community.
This option is not available for a draft (a record not yet shared with a community). For drafts create a link with access permission 'preview', see below.
Proceed as follow to give access to read a record:

  • Select your record in Shared records. Click on the 'Share links' button.
  • Select 'Can view' and click on 'Create a new link'.
  • Click on 'Copy link' and share the link. Anyone with the link can read your record and all its shared versions and download its files.

Can preview - access to read a draft

Proceed as follow to give access to read a draft:

  • Select your draft in My records.
  • Select a community and fill in all the required fields of the upload form. Save the draft.
  • Click on 'Preview' to check you can correctly preview the record.
  • Click on the 'Share links' button.
  • Select 'Can preview' and click on 'Create a new link'.
  • Click on 'Copy link' and share the link. Anyone with the link can read the draft and download its files. If the draft is a new version of a previously shared record, anyone with the link can also read all the other versions of the record.

Can edit - access to edit a draft

Proceed as follow to give access to edit a draft:

  • Select your draft in My records.
  • Select a community and fill in all the required fields of the upload form. Save the draft.
  • Click on the 'Share links' button.
  • Select 'Can edit' and click on 'Create a new link'.
  • Click on 'Copy link' and share the link. Only members of the same community the record is shared with are allowed to edit the draft. If the draft is a new version of a previously shared record or an update of the shared record, with the same link it is possible to also edit all the other versions of the record or create a new version. Only the owner of the record can share the record update or new version with a community.
